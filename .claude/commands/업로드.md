인수: $ARGUMENTS

## 처리 방법

인수 형식은 `탭이름,문서이름` 또는 `탭이름,문서이름,제목` 입니다.

1. `$ARGUMENTS`를 쉼표로 분리해서 파싱하세요.
   - 첫 번째: 탭 이름 (미국주식 / 국내주식 / 암호화폐)
   - 두 번째: 문서 이름 (확장자 없이, 예: SKT)
   - 세 번째(선택): 제목 — 없으면 사용자에게 물어보세요

2. 제목이 인수에 없으면 먼저 이렇게 물어보세요:
   > "제목을 입력해주세요:"

3. 제목을 받은 후 아래 명령어를 Bash로 실행하세요:
   ```
   cd C:/Users/asdf/webtest && python -X utf8 upload_docx.py --tab "탭이름" --file "문서이름" --title "제목"
   ```

4. 실행 결과를 사용자에게 보여주세요.
